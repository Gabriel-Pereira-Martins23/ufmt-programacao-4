# 💻 Programação IV: Desenvolvimento Web (UFMT - CUA)

Este repositório reúne todos os trabalhos práticos, laboratórios e projetos desenvolvidos ao longo da disciplina de **Programação IV** (Desenvolvimento Web) do curso de **Bacharelado em Ciência da Computação** da **Universidade Federal de Mato Grosso (UFMT), Campus Universitário do Araguaia**.

A disciplina aborda de forma prática e aprofundada a arquitetura de sistemas web, dividindo-se entre a construção de interfaces modernas no lado do cliente (Frontend) e o desenvolvimento de regras de negócio, persistência e segurança no lado do servidor (Backend).

---

## 📚 Ementa e Conteúdo Programático

O aprendizado foi estruturado de forma incremental, partindo da estruturação visual básica até a segurança de APIs prontas para produção:

1. **Fundamentos do Frontend (Client-Side):**
   * Estruturação semântica de documentos com **HTML5**.
   * Estilização responsiva, Grid, Flexbox e animações modernas com **CSS3**.
   * Programação assíncrona, manipulação dinâmica da árvore de elementos (DOM) e tratamento de eventos em tempo real com **JavaScript (ES6+)**.
   * Consumo e integração com APIs REST externas (Promises, Fetch API e Async/Await).

2. **Arquitetura e Desenvolvimento Backend (Server-Side):**
   * Fundamentos do framework **NestJS** sob a arquitetura de injeção de dependências (Modules, Controllers e Services).
   * Persistência de dados relacional com banco de dados **SQLite** utilizando o ORM **TypeORM**.
   * Segurança aplicada: criptografia e mascaramento de dados sensíveis com **bcrypt**.
   * Controle de acesso e autenticação por sessão via tokens **JWT (JSON Web Tokens)** integrados ao Passport.

## 📂 Organização dos Trabalhos e Projetos

O repositório está subdividido nos seguintes diretórios estruturais:

### 📔 Laboratórios e Aulas Práticas
* **`AULA_06_07`**: Práticas iniciais de manipulação de DOM e estruturação lógica frontend.
* **`AULA_8`**: Laboratórios focados em lógica de programação web e eventos síncronos.
* **`AULA_09_10`**: Desenvolvimento e blindagem do backend em NestJS, aplicando criptografia de senhas (bcrypt) e guarda de rotas via JWT.

### 🚀 Projetos Temáticos Aplicados
* **`PROJETO_HELLO_KITTY`**: Aplicação interativa focada em layout responsivo avançado, controle de variáveis de estilo CSS e manipulação básica de elementos DOM com temática estilizada.
* **`PROJETO_PEGUE_A_MASCARA`**: Jogo interativo dinâmico que explora temporizadores assíncronos (`setInterval`), lógica de colisão, coordenadas de renderização e manipulação rigorosa de estados de jogo via JS.
* **`PROJETO_POKEDEX`**: Aplicação reativa de consulta que demonstra o domínio em requisições de rede assíncronas assentes na Fetch API, tratando e injetando metadados dinâmicos vindos da API externa PokeAPI.
* **`PROJETO_WIKI_DEMON_SLAYER`**: Portal de informações dinâmico estruturado para consolidar a navegação interna e a organização de layouts complexos baseados em bibliotecas e componentes autorais.
* **`PROJETO_WINX_CLUB`**: Aplicação web desenvolvida para sedimentar os conceitos de estruturação visual estilizada e navegação dinâmica orientada a componentes frontend.
